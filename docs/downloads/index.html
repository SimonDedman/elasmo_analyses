Counts are read live from the same papers_data.json that drives the helper, so this page never goes stale: when papers are ingested and the queue is regenerated, these numbers follow. Open access rows are the quickest wins — those PDFs are free to fetch, they just haven't been collected yet.

For the paywalled tail, the closed-access helpers give four routes per paper (publisher DOI, JSTOR, your library, Google Scholar) across the 1,492 papers known to be closed. Those pages now record progress to the same shared record as the helper, so it doesn't matter which you work from. If you aren't at FIU, set your own library search URL once at the top of any of those pages.
