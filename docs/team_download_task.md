# EEA Shark Review — Team Download Task (closed-access papers)

**Goal:** collect the **1,530 paywalled papers** we can't get automatically. They're grouped **by publisher/journal** so you can claim the ones your institution's library subscribes to (different libraries have different access).

## How to help (5 min to start)

1. **Find a publisher below your library has access to** and put your name in the *Claimed by* column (edit this file, or tell Simon).

2. Open that publisher's page: `outputs/manual_downloads/closed_access/<file>.html` (open in a browser).

3. **Log into your library / VPN / JSTOR / your institution's OneSearch** first.

4. For each paper: click **Download PDF (DOI)**; if you don't have access, try the **JSTOR / OneSearch / Scholar** fallback buttons. Save PDFs to your Downloads folder.

5. Journals are ordered A–Z within each page; clear a whole journal before moving on. Pause every ~25.

6. When done (or you pause), click **⬇ Export routes** on the page and send Simon the JSON + your downloaded PDFs.
