## Technical Notes

### Data Extraction Method

**Year extraction**: Years were extracted from paper_id field using regex pattern `\.(\d{4})\.` (format: `Author.etal.YYYY.Title.pdf`)

**Success rate**: 99.7% (12,207/12,240 papers)

**Missing years**: 33 papers (0.3%) either:
- Lack year in filename
- Have malformed filenames
- Have years outside 1950-2025 range

### Database Source

**Table**: `extraction_log` in `database/technique_taxonomy.db`

**Query**:
```sql
SELECT paper_id FROM extraction_log;
-- Returns: 12,240 papers
```

**Year extraction**:
```python
year_match = re.search(r'\.(\d{4})\.', paper_id)
if year_match:
    year = int(year_match.group(1))
```
